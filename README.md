# EcoTrack
EcoTrack is a simple, intuitive, and smart carbon footprint calculator designed to help users understand and reduce their environmental impact. It tracks emissions from daily activities such as transportation, electricity consumption, food habits, and waste generation, and visualizes them on a clean, interactive dashboard.
